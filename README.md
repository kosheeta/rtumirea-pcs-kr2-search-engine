# rtumirea-pcs-kr2-search-engine
Учебный прототип поискового движка по текстовым файлам на C++.

## Инструкция по сборке

Требования:
- CMake >= 3.31
- C++20-совместимый компилятор (например, `g++`/`clang++`)

Сборка из корня проекта:

```bash
cmake -S . -B build
cmake --build build
```

## Пример запуска

После сборки выполните:

```bash
./build/rtumirea_pcs_kr2_search_engine
```

## Использованные сторонние библиотеки

Сторонние библиотеки в текущей версии не используются.

Проект использует только стандартную библиотеку C++ и систему сборки CMake.
